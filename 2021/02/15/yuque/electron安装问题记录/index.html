electron安装问题记录

发表于 2021-02-15 更新于 2021-03-02

文章迁移～

问题描述

按照 electron 官方的安装教程，先 npm init初始化项目，再使用npm install electron --save-dev进行 electron 的安装。

于是，问题出现了。

问题解决

根据问题的描述，进行了百度。

  • 配置淘宝镜像：npm config set ELECTRON_MIRROR=https://npm.taobao.org/mirrors/electron/或者直接在~/.npmrc中直接添加。
    再次使用命令进行安装，仍报同样的错误。
  • 再次百度，使用淘宝提供的命令进行安装。
    https://npm.taobao.org/mirrors/
    ELECTRON_MIRROR=http://npm.taobao.org/mirrors/electron/ npm install -g electron
    没有了之前的连接超时错误，但是多了个 404 的错误。
  • 继续百度，更换命令
    1. npm install -g package --registry=https://registry.npm.taobao.org 未尝试。
    2. 先安装cnpm, 再安装 electron

成功解决 electron 安装的问题。
