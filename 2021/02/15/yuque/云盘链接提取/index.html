遍历链接

选择合适的网站，例如某资源网、某分享论坛，找到帖子的 ID。例如：

[http://xxx.com/thread-133-1-1.html](http://xxx.com/thread-133-1-1.html)

这个是 Discuz 论坛的帖子链接，133就是帖子的 id。可以将帖子 id 替换为特殊字符，以便快速定位到指定位置，进行 id 的替换。如：

[http://xxx.com/thread-$pageNo$-1-1.html](http://xxx.com/thread-$pageNo$-1-1.html)

然后用通过替换特殊字符串进行字符串的拼接。

拿到链接后，可以直接用requests库进行请求获取。需要注意的是，有些网站有云 waf、需要 header、频繁访问封 IP 等问题，需要注意和解决。

验证链接

百度网盘

  1. 首先判断该链接是否需要密码
  2. 需要密码的进行验证密码，不需要密码的直接获取文件信息
  3. 验证密码后，进行获取文件信息

获取文件信息的链接：[https://pan.baidu.com/share/list?web=5&shorturl=](https://pan.baidu.com/share/list?web=5&shorturl=)【分享的链接后面部分】&root=1

获取用户信息的链接（只能获取用户名称）：[https://pan.baidu.com/pcloud/user/getinfo?query_uk=](https://pan.baidu.com/pcloud/user/getinfo?query_uk=)【分享的链接后面部分】

返回的都是 json 格式，方便处理。

蓝奏网盘

蓝奏网盘相对而言就比较简单，利用 xpath 语句获取即可。需要注意的是，存在两种不同的页面板式，获取信息需要改变一下 xpath 语句。
