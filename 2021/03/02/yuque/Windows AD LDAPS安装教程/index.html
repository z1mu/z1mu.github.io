Windows AD LDAPS安装教程

发表于 2021-03-02

以 Windows server 2012 R2 为例

安装轻型目录服务

打开「添加功能和功能向导」

安装「Active Directory 轻型目录服务」角色

直接下一步、下一步、直至安装完成

创建「AD LDS」实例

填写实例名称

设置端口，可按默认的来

创建程序目录分区


设置文件存储位置， 默认即可

服务账户选择，默认即可

管理员账户选择，默认即可

全部勾选即可

然后一直下一步，等待完成即可

安装证书服务

选择「Active Directory 证书服务」

选择安装的服务，可以全选


一直下一步，直至安装完成

配置 ADDS


选择「证书颁发机构」点击下一步


选择企业


选择「根」


选择「创建新的私钥 」


继续下一步

根据需求进行修改

根据需求进行修改

存储位置，默认即可

继续下一步，完成配置即可

打开「证书颁发机构」


进入「证书模版」管理页面

复制模版

设置模版属性

创建证书模版


选择上一步创建的模版，点击 OK

打开控制台


添加控制单元


申请新证书

一直下一步，选择之前配置的证书模版

注册完成即可

验证 Ldaps

**搜索 ****ldp.exe**

参考下述配置，点击连接

出现下述情况，即为配置成功
